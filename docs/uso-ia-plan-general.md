# Uso de IA en el proyecto

Antes de realizar modificaciones se definió un plan general detallado de trabajo. La IA analizó el código completo y apoyó la limpieza eliminando fragmentos innecesarios o redundantes, optimizando el rendimiento y mejorando la estructura semántica de HTML/CSS/JS. También refinó la estética con criterios de diagramación web y teoría del color.

Tras la planificación se aplicaron mejoras progresivas; cada cambio fue verificado y, cuando surgieron errores (p. ej. en la visualización de imágenes), se corrigieron inmediatamente. El objetivo permanente fue conservar una arquitectura limpia, performante y profesional mientras se avanzaba con el plan asistido por IA.
