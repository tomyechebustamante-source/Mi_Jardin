### Option 2 â€“ Minimal custom volume fader ğŸšï¸

**HTML**

```html
<audio id="player" src="assets/audio/track.mp3" preload="auto"></audio>

<label for="volume">Volume</label>
<input
  id="volume"
  type="range"
  min="0"
  max="1"
  step="0.01"
  value="0.5"
/>

<button id="play">Play / Pause</button>

```

```javascript
<script>
  const audio = document.getElementById('player');
  const volume = document.getElementById('volume');
  const playBtn = document.getElementById('play');

  // Set initial volume
  audio.volume = volume.value;

  // Volume fader
  volume.addEventListener('input', () => {
    audio.volume = volume.value;
  });

  // Simple play/pause button
  playBtn.addEventListener('click', () => {
    if (audio.paused) {
      audio.play();
    } else {
      audio.pause();
    }
  });
</script>

```

**Why this is ideal for your students:**

* Uses only native APIs they should know in 2nd year.

* Shows DOM access, events, and audio properties clearly.

* Easy to restyle the slider with CSS (for the Mars/404 theme).

* No build tools, no framework overhead = perfect for a 2h hackathon.
