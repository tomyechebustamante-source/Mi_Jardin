# Ejercicios de Práctica GSAP

Este directorio contiene archivos de inicio y ejercicios para practicar animaciones GSAP.

## Estructura de Ejercicios

Cada ejercicio incluye:
- Archivos HTML/CSS de inicio
- Instrucciones
- Pistas de solución (cuando están disponibles)

## Cómo Empezar

1. Elige un ejercicio de la lección
2. Copia los archivos de inicio
3. Sigue las instrucciones
4. Prueba tu solución
