  • Use animation-iteration-count: infinite to loop animations indefinitely
  • Performance: Prefer transform and opacity for smooth 60fps
  • Stagger delays: Use animation-delay with nth-child() for sequenced effects

  • Accessibility: Always provide pause controls or respect prefers-reduced-motion
